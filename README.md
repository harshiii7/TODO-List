# 📝 TODO Notes Manager

A simple **TODO / Notes Manager Web Application** built using **Node.js, Express, and EJS**.
This application allows users to create, view, and manage notes easily. Each note is stored as a **text file**, making the project lightweight and easy to understand.

---

## 🚀 Live Demo

🔗 https://todo-list-pyl0.onrender.com

---

## 📌 Features

* Create new notes
* View saved notes
* Read note details
* Edit (rename) notes
* Simple and clean interface
* File-based storage system

## ⚙️ Getting Started

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

git clone https://github.com/harshiii7/TODO-List.git

### 2️⃣ Go to project directory

cd TODO-List

### 3️⃣ Install dependencies

npm install

### 4️⃣ Run the server

node index.js

---

## 💻 Usage

After running the server, open your browser and go to:

http://localhost:4500

You can now create, view, and manage your notes.
